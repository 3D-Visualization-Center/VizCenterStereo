A simple stereo 3D rendering solution for Unity.

Setup:
1) Download this package in unity through the git URL.
2) Navigate to the package in the file explorer. Then open the folder called Runtime.
3) Drag the StereoCanvas prefab from the folder into your scene heirarchy.

That's it! The prefab should be entirely self contained.

NOTE: Please note that if you attempt to test your new stereo implimentation that you must BUILD your project to your device first. It will not appear to work in the editor.
